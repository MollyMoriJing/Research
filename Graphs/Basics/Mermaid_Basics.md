## Layout Direction
graph TD → Top Down
graph LR → Left Right
graph BT → Bottom Top

```mermaid
graph TD
A --> B
B --> C
```

## Node Shapes
[ ] = rectangle
( ) = rounded
(( )) = circle
> ] = rhombus/decision

```mermaid
graph TD
A[Square] --> B(Rounded)
B --> C((Circle))
C --> D>Rhombus]
```
